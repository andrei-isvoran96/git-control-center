# Changelog

## 0.2.0
- Added Killer Differentiators v1:
- Branch Operations Panel (`gitcc.branchActions`).
- Smart Checkout (`gitcc.smartCheckout`) with auto-stash flow.
- One-Click Sync (`gitcc.sync`) with diverged branch guidance.
- Favorite and Recent branches persisted per repository.
- Worktrees view and actions (open/prune/create).
- Actionable error notifications with fix actions.
- Mini Log view with commit actions.
- Command Center (`gitcc.commandCenter`).
- Commit Assist (staged file list, Commit & Sync, post-commit action).
- Branch Compare view (`gitcc.compareBranches` and compare with current).

## 0.1.0
- Initial release of Git Control Center.
- Added repository, branch, changes, commit, and stash views.
- Added Git command palette actions and status bar integration.
- Added parser unit tests and fixture-based coverage.
